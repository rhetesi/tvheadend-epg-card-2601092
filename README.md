# TVHeadend EPG Card

Custom Lovelace card for Home Assistant to display TVHeadend EPG
in a scrollable timeline view.

## Installation

Install via HACS as a custom repository (Frontend).

## Usage

```yaml
type: custom:tvheadend-epg-card
```
